# Workflows Overview

NexusFlow is built around modular workflows that automate content creation.

## Current Workflows

### 📰 News Fetching
Fetches trending topics and real-time content signals.

### ✍️ Blog Generation
Creates long-form SEO optimized blog posts.

### 🖼️ Image Generation
Generates visuals using AI models.

### 🔍 Image Search
Finds relevant images as fallback or enhancement.

---

More workflows will be added as the system evolves.

### 💼 LinkedIn Automation
Generates professional social media posts.

### 🎨 Image Editing
Enhances and modifies generated visuals.
